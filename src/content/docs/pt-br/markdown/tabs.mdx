# Utilizando multiplos códigos em tabs

Podemos mostrar mais de um código ou comandos ao mesmo tempo.

## Import das tabs

Sempre adicione os imports após o título da página.

Podemos usar o seguinte import para começar a usar tabs:

```js
import { Tabs, TabItem } from '@astrojs/starlight/components';
```

`Tabs` é a organização de quantas tabs vão ter.

`TabItem` é os dados de uma das tabs em específico.

## Exemplo de uso de tabs

<br></br>

<Tabs>
<TabItem label="launchSettings.json" icon="seti:json">
```json

  "$schema": "https://json.schemastore.org/launchsettings.json",
```
