# Utilizando Asides

Asides é um plugin que podemos usar para criar notas personalizadas.

Utilizo Asides para apresentar uma informação importante na linguagem desejada.

## Importando Asides no arquivo mdx
<br></br>

Após o título do arquivo mdx, coloque o seguinte import:

```js
import { Aside } from '@astrojs/starlight/components';
```

## Adicionando código

<br></br>

Podemos adicionar código e outros detalhes em markdown dentro de um Aside.

<Aside type="caution" title="Atenção!">
  Podemos adicionar código e outros detalhes utilizando a seguinte estrutura:

  ```js
  import { Aside } from '@astrojs/starlight/components';  
  ```

  Podemos `destacar` uma informação também. 

  Podemos adicionar [Links]() e <span style="color: red">Destacar</span> informações.
</Aside>
