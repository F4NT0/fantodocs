## Como colocar videos no site

- Importe o plugin `VideoPlayer` no seu site:

```js
import { VideoPlayer } from 'starlight-videos/components'
```

- Adicione o video no site:

<VideoPlayer link="https://youtu.be/zwxll667cwM"/>

- Esse é a Tag para adicionar um video solto assim:

```js
<VideoPlayer link="https://youtu.be/zwxll667cwM"/>
```
