## O que é LATEX?

LATEX vem da linguagem TEX onde podemos utilizar fórmulas matemáticas em nossos arquivos .mdx ou markdown.

Github aceita LATEX nativamente, mas para utilizar em nosso site temos que definir algumas estruturas.
