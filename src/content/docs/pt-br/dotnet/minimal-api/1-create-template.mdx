<span style="color: orange">MinimalAPI</span> é uma estrutura nova de requisição para API REST introduzida no .NET 8 que nos auxilia a criar API RESTs mais rápido utilizando uma implementação simples mas eficiente.

Essa nova estrutura ajuda a construir mais rapidamente projetos .NET em microserviços e requer poucas con

## Criando pelo Terminal

:::caution
- Certifique-se que tenha instalado o .NET no seu terminal.
- Para instalar acesse o [Site Oficial](https://dotnet.microsoft.com/en-us/download/dotnet/8.0).
:::

Cheque a versão do seu .NET com o seguinte comando:

```bash title="Verificando versão no Terminal"
dotnet --version
```

Podemos criar facilmente um projeto template de uma MinimalAPI, somente temos que rodar o seguinte comando pelo terminal para poder criar um template:

```bash title="Criando um projeto MinimalAPI template"
dotnet new web -n MinimalApiDemo
```
