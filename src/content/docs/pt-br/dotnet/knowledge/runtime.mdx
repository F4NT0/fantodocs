## Por baixo dos panos

<br></br>

A plataforma .NET é baseada utilizando um _runtime_ chamado <span style="color: lightgreen">CLR</span> (<span style="color: lightgreen">C</span>ommon <span style="color: lightgreen">L</span>anguage <span style="color: lightgreen">R</span>untime) que executa o código compilado em <span style="color: lightgreen">IL</span> (<span style="color: lightgreen">I</span>ntermediate <span style="color: lightgreen">L</span>anguage), gerenciando todos os detalhes necessários pela aplicação como memória, segurança, exceções, etc...
