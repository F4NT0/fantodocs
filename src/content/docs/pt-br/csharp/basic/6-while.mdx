import { Aside } from '@astrojs/starlight/components';

<Aside type="tip" title="Resumo" icon="seti:c-sharp">
  - Continua validando enquanto a condição definida for `verdadeira`.
  - A palavra <span style="color: magenta">while</span> é reservada.
  - Existe apenas uma condição e é verificada `ANTES` de começar.  
</Aside>

Estrutura base de um while:

```csharp title="Incrementador.cs"
var valor = 0;

while (valor <= 5)
{
    Console.WriteLine(valor);
    valor++;
}

// SAÍDA
// 0
// 1
// 2
// 3
// 4
// 5
```

Podemos criar um while que roda infinitamente:

```csharp title="Loop.cs"
var valor = 0;

while (true)
{
    Console.WriteLine(valor);
    valor++;
}

// Vai executar uma adição eternamente
```
