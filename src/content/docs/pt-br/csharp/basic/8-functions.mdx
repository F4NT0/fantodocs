<Aside type="tip" title="Importante!" icon="seti:c-sharp">
  - <span style="color: orange">função</span> e <span style="color: orange">método</span> são quase a mesma coisa.
  - <span style="color: orange">função</span> é o termo mais genérico, descreve qualquer bloco de código que realiza uma tarefa.
  - <span style="color: orange">método</span> é uma função que pertence a uma classe ou objeto.
  - Como C# é uma linguagem orientada a objetos, todas as funções são métodos, porque todos pertencem dentro de uma classe.
</Aside>
