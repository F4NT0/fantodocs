import { Aside } from '@astrojs/starlight/components';

<Aside type="tip" title="Resumo" icon="seti:c-sharp">
    - Utilizado para fazer uma tomada de decisão
    - <span style="color: orange">if</span> é a verificação, valida um conjunto operadores lógicos.
    - Se a verificação lógica for <span style="color: magenta">verdadeira</span> ele faz um conjunto de comandos internos.
    - Caso a verificação lógica for <span style="color: magenta">falsa</span>, ou ele pula para outro if ou entra no bloco else que é usado caso não tenha opções.
</Aside>

Estrutura base:

```csharp

if (condicao) 
{
    // Entra aqui caso a condicao for verdadeira
}
else if (outraCondicao)
{
    // Entra aqui caso a primeira condição for falsa
}
else {
    // Entra aqui caso não tenha mais opções
}

// OBS: podemos colocar quantos else if quisermos

```
