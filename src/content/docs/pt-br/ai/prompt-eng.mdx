## CARE Framework

<span style="color: lightblue">C</span><span style="color: yellow">A</span><span style="color: tomato">R</span><span style="color: lightgreen">E</span> é uma forma estrutural de construir prompts mais efetivos, onde iremos ver os passos a seguir:

<span style="color: lightblue">Context</span> → Define a cena, explique o background do problema, diga para a IA detalhes que ache util para pensar no problema.

<span style="color: yellow">Ask</span> → Seja claro no que deseja, que tipo de resposta está esperando da IA.

<span style="color: tomato">Rules</span> → Define os limites desejados, define o tom, tamanho, estilo e audiencia esperada para a resposta.

<span style="color: lightgreen">Example</span> → Se saiba de um exemplo que possa ajudar a IA a pensar em melhores resultados, escreva eles para a IA, quanto mais exemplos melhor para o resultado.

### Exemplo

Este é um prompt bem vago:

```md
Me ajude a construir uma mensagem de atualização para o meu time
```

Este é um prompt utilizando o padrão CARE:

<span style="color: lightblue">Você está ajudando um lider de projeto a comunicar um update para times remotos.</span> <span style="color: yellow">Escreva uma mensagem amigável e concisa</span> <span style="color: tomato">que tenha no máximo 100 palavras sumarizando no timeline do projeto.</span> <span style="color: lightgreen">Use um tom similar como "Rápido update para todos - aqui está o que foi mudado..."</span>.
